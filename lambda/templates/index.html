Unfriended

Check who unfriended you on Facebook!

This app is now a read-only archive.

Unfriended was a Flask app that tracked your Facebook friends list over time and showed you who had unfriended you. Users would log in with Facebook OAuth, upload their friends-list HTML (after Facebook removed API access to friends lists), and the app would compare the current list against stored data to find removed friends.

Facebook API changes made the core functionality impossible, so the app has been preserved here as an archive.
